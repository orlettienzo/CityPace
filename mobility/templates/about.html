La team CityPace

Citypace est un projet créé par 5 étudiants de l'UCLouvain dans le cadre du cours d'LINFO1002. Le véritable but de ce projet est d'appendre à utiliser Flask, SQLite et l'html/css ce qui équivaut donc à créer un site web utilisant sur une base de données.

Nos objectifs pour ce site

Nous tenons à faire de ce site un moyen facile et efficace de consulter les données proposées par Telraam (expliqué ci dessous). Nous voulons également rendre ce site authentique afin de se demarquer des autres. Ce site ce veut aussi pratique pour les études effectué depuis ces données.

D'où viennent ces données?

CityPace utilise les données des capteurs Telraam. Telraam a un réseau de capteurs sous la forme d'une caméra collée à une fenêtre qui donne sur la rue et un algorithme de vision par ordinateur pour compter le nombre de voitures, piétons, cyclistes et poids lourds. Toutes les heures, le capteur renvoie le nombre d'objets détectés et leurs vitesses agrégées à Telraam.net.

CityPace n'utilise pas l'api Telraam directement mais une copie sous la forme d'un csv converti en base de données SQL.
